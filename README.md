#  Sistema de Chamados

Sistema simples para gerenciamento de chamados de suporte técnico.

---

##  Funcionalidades
- Criar chamados
- Listar chamados
- Marcar chamados como resolvidos

---

##  Tecnologias
- HTML
- CSS
- JavaScript

---

##  Objetivo
Projeto desenvolvido para simular um sistema real de suporte técnico, permitindo o registro e acompanhamento de chamados.
